# Results

The final goal to be achieved within the timeframe of fifty years is defined as a predominant use of the audio focused mobile technology.
This technology in its final form entails an implant, which is able to stimulate the audio reception of users as well as implanted sensors in one hand, which are able to detect touch.
Users can interact with the device with spoken word or lorming, which is detected via the hand sensors.
Artificial Intelligence enables natural feeling interaction with the device, which is able to carry out a multitude of functionalities currently enabled by smartphones.
Examples are, receiving and sending messages and calls, searching for information, listening to music, audio books, podcasts and other audio input formats as well as setting alarms and reminders.
Many more use cases are feasible to be supported by the technology.

## Steps to be taken

Achieving this goal relies on several dimensions of change, which have been identified using the input of the conducted student workshop.
Each aspect should be aided by a specific roadmap, while recognising their interdependencies.

### Learning lorming

In order to use the technology to its full potential, mastering lorming is a key aspect.
This ensures a fast and non-intrusive mean of input.
Therefore learning lorming is one of the key challenges to address, when rolling out the technology.
